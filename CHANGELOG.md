## [0.1.0-auth-foundation] - 2026-07-08

### Added
- Monorepo directory foundation (`apps/web`, `apps/mobile`, `packages/shared`, `supabase/migrations`, `docs`).
- Comprehensive project documentation in `docs/` (`PRODUCT_VISION.md`, `MVP_SCOPE.md`, `AUTH_FLOW.md`, `GITHUB_WORKFLOW.md`, `SECURITY_CHECKLIST.md`, `ROADMAP.md`, drafts of terms/privacy, and QA checklist).
- Next.js Web App with full authentication pages and responsive dashboard placeholder.
- Flutter Mobile App authentication screens and Routing Gate.
- Supabase SQL migration for automatic profile creation and Row Level Security.
- GitHub Actions CI build check workflows for both web and mobile.
